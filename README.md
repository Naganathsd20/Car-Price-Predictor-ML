# 🚘 AutoWorth

🚘 AutoWorth is a Machine Learning web application that predicts the estimated resale value of used cars based on user inputs such as car company, model, manufacturing year, fuel type, and kilometers driven.

---

## 📖 Project Overview 

🚘 AutoWorth uses a **Linear Regression Machine Learning model** to estimate the resale price of used cars. The application is built using **Flask** for the backend and **HTML, CSS, JavaScript, and jQuery** for the frontend, providing users with an easy-to-use web interface for instant predictions.

---

## ✨ Features

- 🚗 Predict used car resale prices
- 🤖 Machine Learning based prediction
- 🌐 Flask web application

## 🚀 Installation

### Clone the Repository

```bash
git clone https://github.com/Naganathsd20/Car-Price-Predictor-ML.git
```

### Navigate to the Project Directory

```bash
cd Car-Price-Predictor-ML
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
python application.py
```

### Open in Your Browser

```
http://127.0.0.1:5000
```
